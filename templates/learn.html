{{ 'What is Tuberculosis?' if lang == 'en' else 'Apa itu Tuberkulosis?' }}

{% if lang == 'en' %} Tuberculosis (TB) is an infectious disease caused by the bacterium Mycobacterium tuberculosis. It primarily affects the lungs but can also impact other parts of the body. TB spreads through airborne particles when an infected person coughs, sneezes, or speaks. {% else %} Tuberkulosis (TB) adalah penyakit menular yang disebabkan oleh bakteri Mycobacterium tuberculosis. Penyakit ini terutama menyerang paru-paru tetapi juga dapat berdampak pada bagian tubuh lainnya. TB menyebar melalui partikel udara ketika orang yang terinfeksi batuk, bersin, atau berbicara. {% endif %}

{% if lang == 'en' %} TB remains one of the world's deadliest infectious diseases, but it is both preventable and curable with proper treatment. Early detection through chest X-rays and other diagnostic tools is crucial for successful treatment outcomes. {% else %} TB tetap menjadi salah satu penyakit menular paling mematikan di dunia, tetapi dapat dicegah dan disembuhkan dengan pengobatan yang tepat. Deteksi dini melalui X-ray dada dan alat diagnostik lainnya sangat penting untuk hasil pengobatan yang sukses. {% endif %}

{{ 'Lung Infection Progression' if lang == 'en' else 'Perkembangan Infeksi Paru-paru' }}

{{ 'Stage 1: Exposure' if lang == 'en' else 'Tahap 1: Paparan' }}

{{ 'TB bacteria enter the lungs through inhalation of infected droplets' if lang == 'en' else 'Bakteri TB masuk ke paru-paru melalui menghirup tetesan yang terinfeksi' }}

{{ 'Stage 2: Latent TB' if lang == 'en' else 'Tahap 2: TB Laten' }}

{{ 'Bacteria are present but inactive. No symptoms, not contagious' if lang == 'en' else 'Bakteri hadir tetapi tidak aktif. Tidak ada gejala, tidak menular' }}

{{ 'Stage 3: Active TB' if lang == 'en' else 'Tahap 3: TB Aktif' }}

{{ 'Bacteria multiply and damage lung tissue. Symptoms appear, highly contagious' if lang == 'en' else 'Bakteri berkembang biak dan merusak jaringan paru-paru. Gejala muncul, sangat menular' }}

{{ 'Stage 4: Recovery' if lang == 'en' else 'Tahap 4: Pemulihan' }}

{{ 'With treatment, bacteria are eliminated and lung tissue heals over 6+ months' if lang == 'en' else 'Dengan pengobatan, bakteri dieliminasi dan jaringan paru-paru sembuh selama 6+ bulan' }}

{{ 'Common Symptoms' if lang == 'en' else 'Gejala Umum' }}

{{ 'Active TB infection typically presents with the following symptoms:' if lang == 'en' else 'Infeksi TB aktif biasanya menunjukkan gejala berikut:' }}

  • {{ 'Persistent cough lasting 3+ weeks (may produce blood)' if lang == 'en' else 'Batuk persisten selama 3+ minggu (mungkin mengeluarkan darah)' }}
  • {{ 'Chest pain or discomfort when breathing or coughing' if lang == 'en' else 'Nyeri dada atau ketidaknyamanan saat bernapas atau batuk' }}
  • {{ 'Unexplained weight loss and loss of appetite' if lang == 'en' else 'Penurunan berat badan yang tidak dapat dijelaskan dan kehilangan nafsu makan' }}
  • {{ 'Fever, chills, and night sweats' if lang == 'en' else 'Demam, menggigil, dan keringat malam' }}
  • {{ 'Extreme fatigue and weakness' if lang == 'en' else 'Kelelahan ekstrem dan kelemahan' }}

⚠️ {{ 'Important:' if lang == 'en' else 'Penting:' }} {{ 'If you experience these symptoms, especially a persistent cough, seek medical attention immediately. Early diagnosis is crucial for effective treatment.' if lang == 'en' else 'Jika Anda mengalami gejala ini, terutama batuk yang terus-menerus, segera cari pertolongan medis. Diagnosis dini sangat penting untuk pengobatan yang efektif.' }}

{{ 'Prevention Tips' if lang == 'en' else 'Tips Pencegahan' }}

{{ 'Protect yourself and others from TB with these prevention strategies:' if lang == 'en' else 'Lindungi diri Anda dan orang lain dari TB dengan strategi pencegahan ini:' }}

{{ 'For Everyone:' if lang == 'en' else 'Untuk Semua Orang:' }}

  • {{ 'Get vaccinated with BCG (Bacillus Calmette-Guérin) vaccine, especially in high-risk areas' if lang == 'en' else 'Dapatkan vaksinasi BCG (Bacillus Calmette-Guérin), terutama di daerah berisiko tinggi' }}
  • {{ 'Maintain good ventilation in living and working spaces' if lang == 'en' else 'Pertahankan ventilasi yang baik di ruang tinggal dan kerja' }}
  • {{ 'Cover your mouth when coughing or sneezing' if lang == 'en' else 'Tutup mulut Anda saat batuk atau bersin' }}
  • {{ 'Maintain a healthy immune system through proper nutrition and exercise' if lang == 'en' else 'Pertahankan sistem kekebalan tubuh yang sehat melalui nutrisi dan olahraga yang tepat' }}

{{ 'If You Have TB:' if lang == 'en' else 'Jika Anda Memiliki TB:' }}

  • {{ 'Complete your full course of treatment as prescribed' if lang == 'en' else 'Selesaikan seluruh pengobatan sesuai resep' }}
  • {{ 'Isolate yourself during the first 2-3 weeks of treatment' if lang == 'en' else 'Isolasi diri Anda selama 2-3 minggu pertama pengobatan' }}
  • {{ 'Wear a mask when around others' if lang == 'en' else 'Kenakan masker saat berada di sekitar orang lain' }}
  • {{ 'Avoid close contact with vulnerable populations (children, elderly, immunocompromised)' if lang == 'en' else 'Hindari kontak dekat dengan populasi rentan (anak-anak, lansia, immunocompromised)' }}

{{ 'Treatment Options' if lang == 'en' else 'Opsi Pengobatan' }}

{{ 'TB is curable with proper antibiotic treatment. The standard treatment involves:' if lang == 'en' else 'TB dapat disembuhkan dengan pengobatan antibiotik yang tepat. Pengobatan standar meliputi:' }}

{{ 'Drug-Susceptible TB:' if lang == 'en' else 'TB yang Sensitif terhadap Obat:' }}

  • {{ '6-month treatment regimen with multiple antibiotics' if lang == 'en' else 'Regimen pengobatan 6 bulan dengan beberapa antibiotik' }}
  • {{ 'First-line drugs: Isoniazid, Rifampicin, Ethambutol, Pyrazinamide' if lang == 'en' else 'Obat lini pertama: Isoniazid, Rifampisin, Etambutol, Pirazinamid' }}
  • {{ 'Daily medication adherence is critical for success' if lang == 'en' else 'Kepatuhan minum obat harian sangat penting untuk kesuksesan' }}

{{ 'Drug-Resistant TB:' if lang == 'en' else 'TB yang Resisten terhadap Obat:' }}

  • {{ 'Longer treatment duration (9-20 months)' if lang == 'en' else 'Durasi pengobatan yang lebih lama (9-20 bulan)' }}
  • {{ 'Second-line drugs with potential side effects' if lang == 'en' else 'Obat lini kedua dengan potensi efek samping' }}
  • {{ 'Requires close medical supervision' if lang == 'en' else 'Memerlukan pengawasan medis yang ketat' }}

⚠️ {{ 'Critical:' if lang == 'en' else 'Kritis:' }} {{ 'Never stop treatment early, even if you feel better. Incomplete treatment can lead to drug-resistant TB, which is much harder to cure.' if lang == 'en' else 'Jangan pernah menghentikan pengobatan lebih awal, bahkan jika Anda merasa lebih baik. Pengobatan yang tidak lengkap dapat menyebabkan TB yang resisten terhadap obat, yang jauh lebih sulit disembuhkan.' }}
